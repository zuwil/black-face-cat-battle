# 黑臉貓大作戰 — 遊戲需求文件

> 最後更新：2026-04-01

---

## 基本資訊

- **遊戲名稱**：黑臉貓大作戰
- **類型**：縱向卷軸射擊遊戲
- **平台**：網頁遊戲（HTML5 Canvas 2D）
- **技術**：純原生 JavaScript + Canvas API，零依賴
- **畫布大小**：480 x 640（直式）
- **部署**：GitHub Pages — https://zuwil.github.io/black-face-cat-battle/

---

## 角色設計

### 玩家 — 黑臉貓咪
- 臉部黑色的可愛貓咪，像素風格
- 鍵盤方向鍵 / WASD 移動
- 手機觸控：手指拖動移動
- 自動射擊貓掌波
- 3 條命，被擊中後閃爍無敵 2 秒
- 碰撞箱比精靈稍小（提升手感）

### 貓掌波 — 3 段升級系統
| 等級 | 效果 |
|------|------|
| Lv.1 | 單發貓掌，直線往上 |
| Lv.2 | 雙發貓掌，稍微分散 |
| Lv.3 | 三發貓掌，扇形展開，掌印更大 |

- 升級方式：擊敗雜兵有機率掉落升級道具（發光肉球圖示）
- 被擊中時降一級（最低 Lv.1）

### 雜兵 — 小貓小狗混合
- **小狗**：直線往下飛，偶爾左右晃動，偶爾往下丟慢速子彈
- **小貓**：Z 字形移動，5 種射擊模式隨機分配：
  - aimed（瞄準玩家）
  - spread3（三連散射）
  - burst（連發 3 次）
  - ring（6 方向環射）
  - wave（雙發平行）

### Boss — 胖博美
- 外觀：胖、憨、呆的棕色博美犬，下巴有黑色區塊
- 出場條件：擊敗 30 隻雜兵後出場
- **階段 1**：左右緩慢移動，發射大骨頭彈幕（3 發散射）
- **階段 2**（血量 < 50%）：加速移動，追加往下衝撞攻擊，彈幕增加到 5 發
- 擊敗後顯示勝利畫面（博美翻肚皮）

---

## 道具系統

- 雜兵被擊敗時有 15% 機率掉落貓掌升級道具
- 道具外觀：發光的肉球圖示（脈衝閃爍效果）
- 拾取後貓掌波升一級（最高 Lv.3）

---

## 遊戲流程

```
標題畫面 → 遊戲中（雜兵波次）→ Boss 戰 → 勝利/失敗畫面
```

### 標題畫面
- 顯示遊戲名稱「黑臉貓大作戰」
- 黑臉貓咪 + 胖博美的像素圖展示
- 「按任意鍵開始」提示
- 「[ ] 鍵切換背景色溫」提示

### 遊戲中 HUD
- 左上：分數
- 右上：剩餘生命（小貓頭圖示 x3）
- 左下：目前貓掌等級（Lv.1/2/3）
- Boss 戰時：上方中央顯示 Boss 血量條（綠 > 黃 > 紅）

### 勝利畫面
- 胖博美翻肚皮倒下
- 顯示最終分數
- 按任意鍵回到標題

### 失敗畫面
- 「Game Over」文字
- 最終分數
- 按任意鍵回到標題

## 美術風格

- 像素可愛風（Pixel Art）
- 所有角色像素圖直接用 Canvas API 繪製
- 大眼睛、圓臉蛋、鮮明色彩
- 敵方子彈為亮桃紅色 + 發光效果，確保可見度

---

## 背景系統

- 像素風天空 + 雲朵，持續向下捲動營造飛行感
- 5 種色溫可切換（按 `[` / `]` 鍵）：
  - 夜空（深藍紫）
  - 陰天（灰色調）
  - 晴天（天藍，預設）
  - 暖陽（橘色調）
  - 夕陽（紅橘漸層）

---

## 操作方式

### 電腦
- **方向鍵 / WASD** — 移動
- **自動射擊**
- **[ / ]** — 切換背景色溫

### 手機
- **點擊畫面** — 開始 / 重新開始
- **手指拖動** — 移動黑臉貓
- **自動射擊**
- 畫面自動縮放適應螢幕

---

## 技術需求

- 純 HTML5 Canvas 2D，無框架依賴
- ES modules 結構
- 60fps 遊戲迴圈（requestAnimationFrame）
- Node.js 原生測試（node --test）
- GitHub Pages 靜態部署
