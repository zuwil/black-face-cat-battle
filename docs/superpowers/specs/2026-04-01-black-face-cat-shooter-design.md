# 黑臉貓大作戰 — 遊戲設計規格書

## 概要

一款網頁端縱向卷軸射擊遊戲，使用純 HTML5 Canvas 2D 開發，像素可愛風美術。玩家操控一隻黑臉貓咪，發射貓掌波擊敗各種小貓小狗雜兵，最終挑戰一隻胖、憨、呆的棕色博美 Boss。

## 技術方案

- **平台**：網頁遊戲（HTML5 Canvas 2D）
- **框架**：無，純原生 JavaScript + Canvas API
- **畫布大小**：480 x 640（直式）
- **遊戲迴圈**：`requestAnimationFrame`，60fps 邏輯更新
- **每幀流程**：輸入處理 → 更新位置 → 碰撞偵測 → 渲染繪製

## 檔案結構

```
index.html          — 入口頁面 + Canvas
css/style.css       — 樣式
js/
  main.js           — 遊戲初始化、遊戲迴圈
  player.js         — 黑臉貓咪（移動、射擊、升級）
  enemy.js          — 雜兵（小貓、小狗）
  boss.js           — 胖博美 Boss
  bullet.js         — 貓掌波子彈（3 級）
  powerup.js        — 升級道具
  collision.js      — 碰撞偵測
  sprites.js        — 像素圖繪製（用 Canvas 直接畫）
  game-state.js     — 遊戲狀態管理
```

## 角色設計

### 玩家 — 黑臉貓咪

- 臉部黑色的可愛貓咪，像素風格
- 鍵盤方向鍵 / WASD 移動，自動射擊（或按空白鍵射擊）
- 3 條命，被擊中後閃爍無敵 2 秒
- 碰撞箱比精靈稍小（提升手感）

### 貓掌波 — 3 段升級系統

| 等級 | 效果 |
|------|------|
| Lv.1 | 單發貓掌，直線往上 |
| Lv.2 | 雙發貓掌，稍微分散 |
| Lv.3 | 三發貓掌，扇形展開，掌印更大 |

- 升級方式：擊敗雜兵有機率掉落升級道具（發光肉球圖示）
- 被擊中時降一級（最低 Lv.1）

### 雜兵

- **小狗**：直線往下飛，偶爾左右晃動，不發射子彈
- **小貓**：Z 字形移動，會往玩家方向吐毛球

### Boss — 胖博美

- 外觀：胖、憨、呆的棕色博美犬，下巴有黑色區塊
- 出場條件：擊敗 30 隻雜兵後出場
- **階段 1**：左右緩慢移動，發射大骨頭彈幕
- **階段 2**（血量 < 50%）：加速移動，追加往下衝撞攻擊
- 擊敗後顯示勝利畫面

## 道具系統

- 雜兵被擊敗時有一定機率掉落貓掌升級道具
- 道具外觀：發光的肉球圖示
- 拾取後貓掌波升一級（最高 Lv.3）

## 遊戲流程

```
標題畫面 → 遊戲中（雜兵波次）→ Boss 戰 → 勝利/失敗畫面
```

### 標題畫面

- 顯示遊戲名稱「黑臉貓大作戰」
- 黑臉貓咪 + 胖博美的像素圖展示
- 「按任意鍵開始」提示

### 遊戲中 HUD

- 左上：分數
- 右上：剩餘生命（小貓頭圖示 x3）
- 左下：目前貓掌等級（Lv.1/2/3）

### 背景

- 像素風天空 + 雲朵，持續向下捲動營造飛行感

### 勝利畫面

- 胖博美翻肚皮倒下
- 顯示最終分數

### 失敗畫面

- 「Game Over」文字
- 最終分數
- 「重新開始」按鈕

## 美術風格

- 像素可愛風（Pixel Art）
- 所有角色像素圖直接用 Canvas API 繪製（`sprites.js`）
- 大眼睛、圓臉蛋、鮮明色彩
